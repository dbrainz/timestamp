DBrainz's FCC API Timestamp Service

Pass in a date either as a natural language date or a unix timestamp and both a unix timestamp and natural date will be returned. If a valid date or timestamp is not passed in, null will be returned for both values.

"https://dbrainz-timestamp.herokuapp.com/June%206,%202006" and
"https://dbrainz-timestamp.herokuapp.com/1149552000000" will both return
{"unix":1149552000000,"natural":"June 6, 2006"}
